#  Микро-игра Платформер
Данный проект представляет собой микроигру в жанре "Платформер" с 2 уровнями.


## Установка
Стандартная процесс, скачивайте репозиторий и запускаете проект.
Проект реализован на движке версии 2022.3.20F1

## Использование
В папке Asset-> Scenes находятся 2 сцены. 
Каждая сцена представляет из себя 1 уровень. 
Можете открывать любую сцену по желанию.
Управление стандартное - WASD , пробел-прыжок.
